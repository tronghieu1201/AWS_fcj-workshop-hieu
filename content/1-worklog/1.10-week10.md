## 3. Content

- Amazon Cognito:
  - Learned how to authenticate users in a serverless application.
  - Practiced creating a User Pool and configuring sign-up and sign-in.
  - Became familiar with authentication tokens and how to protect APIs.

- SQS and SNS:
  - Learned how to process asynchronous tasks using queues and notifications.
  - Practiced using SQS to store messages waiting to be processed.
  - Learned how SNS sends notifications to multiple subscribers.
  - Combined Lambda with SQS/SNS to process event-driven workflows.

- CloudWatch and X-Ray:
  - Learned how to monitor a serverless application in a production environment.
  - Practiced viewing logs, metrics, and errors in CloudWatch.
  - Became familiar with AWS X-Ray to analyze request flow and processing time.

- AWS Step Functions:
  - Learned how to orchestrate multi-step workflows on AWS.
  - Practiced creating a state machine to connect multiple processing steps.
  - Understood how Step Functions helps manage complex workflows more clearly.

- Personal Project - Route 53 Custom Domain:
  - Configured a Route 53 Hosted Zone for the personal domain.
  - Created records to point the domain to the CloudFront Distribution.
  - Tested the full portfolio flow from domain, CloudFront, S3, API Gateway, Lambda, to DynamoDB.
