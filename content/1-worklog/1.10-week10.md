## 3. Content

- Amazon Cognito:
  - Learned how to authenticate users in a serverless application.
  - Practiced creating a User Pool and configuring basic sign-up and sign-in flows.
  - Became familiar with authentication tokens and how they can be used to protect APIs.

- SQS and SNS:
  - Learned how to process asynchronous tasks using queues and notifications.
  - Practiced using SQS to store messages waiting to be processed.
  - Learned how SNS can be used to send notifications to multiple subscribers.
  - Combined Lambda with SQS/SNS to process event-driven workflows.

- CloudWatch and X-Ray:
  - Learned how to monitor a serverless application in a production environment.
  - Practiced viewing logs, metrics, and application errors in CloudWatch.
  - Became familiar with AWS X-Ray for analyzing request flows and processing time.

- AWS Step Functions:
  - Learned how to orchestrate multi-step workflows on AWS.
  - Practiced creating a state machine to connect multiple processing steps.
  - Understood how Step Functions makes complex workflows easier to manage and monitor.

- Personal project - Final HTTPS and deployment validation:
  - Reviewed the final portfolio source code on GitHub.
  - Checked the final commit and deployment-related files.
  - Verified the production build files before deployment.
  - Checked that the CloudFront Distribution was enabled and ready to serve the website.
  - Tested the CloudFront HTTPS URL to confirm that the portfolio homepage loaded successfully.
  - Checked and fixed the CloudFront error behavior when refreshing or accessing project detail pages directly.
  - Verified the final portfolio flow, including the homepage, project detail page, API Gateway, Lambda, and DynamoDB visitor counter.
