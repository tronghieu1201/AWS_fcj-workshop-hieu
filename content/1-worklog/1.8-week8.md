## 3. Content

- AWS Backup:
  - Learned how to create a backup plan for AWS resources using AWS Backup.
  - Practiced configuring a backup plan, backup vault, and backup rule.
  - Became familiar with selecting resources for backup and checking backup jobs.

- SSL/HTTPS for Application:
  - Learned how to use Route 53, ACM, and CloudFront to configure HTTPS.
  - Practiced issuing an SSL certificate and attaching it to a CloudFront Distribution.
  - Configured website access through HTTPS to improve security.

- Docker and Amazon ECR:
  - Learned the basic concepts of Docker, image, and container.
  - Practiced packaging an application into a Docker image.
  - Pushed the image to Amazon ECR to prepare for AWS deployment.

- Amazon ECS:
  - Learned how to run containers on Amazon ECS.
  - Practiced creating an ECS Cluster, Task Definition, and Service.
  - Verified that the containerized application was running on ECS.

- Personal Project - CI/CD:
  - Created a GitHub Actions workflow to automatically deploy the portfolio to S3.
  - Configured limited IAM permissions for the deployment process.
  - Verified that the website was updated after each code push.
