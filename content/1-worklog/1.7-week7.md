- IAM Permission Boundary:
  - Learned how to limit the maximum permissions of an IAM User using Permission Boundary.

- IAM Role & Condition:
  - Reviewed IAM User, Group, Role, and the Assume Role process.

  - Enabled Security Hub to check the security status of the AWS account.

  - Practiced configuring managed rules, custom rules, and testing blocked/allowed requests.
