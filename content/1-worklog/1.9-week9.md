## 3. Content

- CI/CD for a Serverless Application:
  - Learned the automated build and deployment process for a serverless application.
  - Became familiar with CodePipeline, CodeBuild, and automated deployment workflows.
  - Practiced configuring a pipeline to reduce manual deployment steps.

- Lambda, S3, and DynamoDB:
  - Learned how AWS Lambda handles backend logic in a serverless architecture.
  - Practiced making Lambda interact with S3 and DynamoDB.
  - Became familiar with storing data and handling events without managing servers.

- API Gateway and Frontend:
  - Learned how to create an API endpoint using Amazon API Gateway.
  - Connected a frontend application to API Gateway to call a Lambda function.
  - Configured CORS so the frontend could call the API correctly.

- AWS SAM:
  - Learned how to use AWS SAM to deploy a serverless application with a template.
  - Practiced defining Lambda, API Gateway, and DynamoDB resources in a configuration file.
  - Deployed a serverless application through SAM instead of manually creating each service.

- Personal project - S3 and CloudFront:
  - Built the production version of the portfolio project.
  - Checked the generated `dist` folder before deployment.
  - Uploaded the production build files to an Amazon S3 bucket.
  - Configured the S3 bucket for static website hosting.
  - Created a CloudFront Distribution to deliver the portfolio website more reliably.
  - Tested the CloudFront domain to confirm that the portfolio website worked after deployment.
